# Toolbox 🛠️

A collection of utilities, scripts, and developer tools that make life easier.

## Philosophy

Every developer accumulates scripts and tools over time. This is where mine live — organized, documented, and ready to use.

## Categories

- **/scripts/** — Shell/Python scripts for automation
- **/cli/** — Command-line utilities
- **/config/** — Dotfiles and configuration templates
- **/snippets/** — Reusable code snippets

## Usage

Each tool includes its own README with setup and usage instructions.
